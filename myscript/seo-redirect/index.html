1) When do you want to use this?

When you are migrating to the new version of your website and you will have other URLs, it is best practice to redirect (301) old URLs to new URLs to keep the search engines positions.

2) What exactly it does?

You need a list of URLs from your old and your new website. On Wordpress, you can use ðŸ”— Yoast SEO plugin to generate a sitemap. From sitemap, simply copy the URLs - paste Old URLs to the left pane, and New URLs to the right pane. If you are using the same name of posts and pages, but different folder/URL structure, this tool will find the pairs and export it to CSV.

3) How to make it work?

Once you copy/pasted URLs you need to select which part of it will be used as string for searching and pairing with new URL. Type in how many characters at the start of the URL will be taken off (ie. http://www.domain.com/ = 22) and how many from the end (ie. .html = 5)

Hint: http://www.domain.com/contact.html => contact

4) Finished. Now what?

When completed, you can download CSV with Old - New URL pairs. Where empty, exact pair was not found. You will have to assign it manually. But most of them will be done automatically.
